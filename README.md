# TradePulse — Trading Journal

A comprehensive trading journal for stocks, options, and futures. Track trades, review performance, manage playbooks, and grow as a trader.

## Features

- **Dashboard** — P&L charts, calendar heatmap, stat cards, account tracking
- **Journal** — Daily pre-market planning, end-of-day reviews, mood & market condition tracking
- **Goals** — Daily % targets, compounding projections, streak tracking
- **Holdings** — Open position tracking with live price lookup and unrealized P&L
- **Review** — Risk management analysis, pattern insights, trade replay
- **Playbook** — 38 built-in strategies + custom setups with performance tracking
- **Wheel** — Dedicated wheel strategy tracker (CSP → CC cycle)
- **Watchlist** — Multi-list watchlist with price alerts and notes
- **Trade Log** — Sortable, filterable trade history with CSV import/export
- **Settings** — Custom fields, accounts, futures presets, dark/light theme, tab reordering

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Deploy

Push to GitHub and connect to [Vercel](https://vercel.com) for instant deployment.

## Tech Stack

- React 18
- Recharts (charts)
- Lucide React (icons)
- Vite (build tool)
- localStorage (data persistence)
